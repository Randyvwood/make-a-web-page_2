The Web -
The world wide web is a collection of HTML documnets.
The major pieces of the of the web are you (pc/browser), the net and severs.

HTML -
HTML is an acronym for hyper text markup language.
HTML doucumnes are made up of text content, and HTML attributes.

Text content is what you see.
Markup is what it looks like and is created using html tags.
HTML attributes include refrences and links to other documents (music, video, etc.)

Tags-
There are several types of tags including attribute tags, inline tags and block tags.

Attribute tags include links to other pages like My Facebook. As well as images and videos seen on the page.
Inline tags include Bold, Italics, and span.
Block tags include the paragraph tag and the div tag.

Lesson Two

CSS -
Cascading Style Sheets - style sheet language used for describing the look and formatting of a
document written in a markup language.

DOM -
Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are
organized in a tree structure, called the DOM tree.
